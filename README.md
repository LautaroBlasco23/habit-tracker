# Habit Tracker

This fictitional app lets you save all your repetitive daily tasks.

Using habit tracker, you can know which tasks you've done in the past days. and which ones you will do in the following days.

# Habit service development and testing 

Inside "/server/habits/" there's a Makefile that contains some usefull scripts to test and start the API in your machine.

TL;DR -> run "run-app" script and the API and it's database will be running.
